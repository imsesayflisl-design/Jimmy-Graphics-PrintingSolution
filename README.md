# Web3 Design Agency Website

A modern, fully functional Web3 Design Agency website built with HTML5, CSS3, and Vanilla JavaScript.

## Features

✨ **Modern Design**
- Dark theme with vibrant gradient accents (Pink, Purple, Cyan)
- Responsive design that works on all devices
- Smooth animations and transitions

📱 **Pages Included**
- **Home** (home.html) - Hero section, services overview, testimonials, stats
- **About** (about.html) - Company information, values, team
- **Services** (services.html) - Detailed service offerings
- **Portfolio** (portfolio.html) - Showcase of completed projects
- **Contact** (contact.html) - Contact form and information

🎨 **Components**
- Navigation bar with mobile hamburger menu
- Hero section with animated 3D element
- Service cards and features
- Testimonial carousel
- Statistics section
- Contact form with validation
- Footer with social links

⚡ **Performance**
- No external dependencies (vanilla JavaScript)
- Optimized CSS with CSS variables
- Mobile-first responsive design
- Smooth scroll navigation

## How to Use

1. **Open in Browser**: Double-click `home.html` to open the website
2. **Local Server** (Recommended):
   ```bash
   # Using Python 3
   python -m http.server 8000
   
   # Using Python 2
   python -m SimpleHTTPServer 8000
   ```
   Then visit: `http://localhost:8000`

## Customization

### Change Colors
Edit the CSS variables in `css/style.css` (lines 3-13):
```css
:root {
  --primary-pink: #ff1493;
  --primary-purple: #a855f7;
  /* ... more colors ... */
}
```

### Update Content
Simply edit the HTML files with your own text, images, and information.

### Add Images
Place your images in the `images/` folder and reference them in HTML:
```html
<img src="images/your-image.png" alt="Description">
```
