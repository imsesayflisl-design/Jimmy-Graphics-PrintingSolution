## 🔧 Customization Guide

### Change Logo/Branding
Edit `home.html` line 15:
```html
<a href="home.html" class="logo">We3</a>  <!-- Change "We3" to your name -->
```

### Update Colors
Edit `css/style.css` lines 3-13:
```css
:root {
  --primary-pink: #ff1493;      /* Change these color codes */
  --primary-purple: #a855f7;
  --dark-bg: #0f0f0f;
  /* ... etc ... */
}
```

### Modify Content
- Edit text in any `.html` file
- Update contact information in footer
- Add your actual project images to `images/` folder
- Update service descriptions in `services.html`

### Add Images
Place your images in the `images/` folder and reference:
```html
<img src="images/your-image.png" alt="Description">
```
